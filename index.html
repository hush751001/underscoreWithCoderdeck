UNDERSCORE.JS

Introduction

Underscore is a utility-belt library for JavaScript that provides a lot of the functional programming support that you would expect in Prototype.js (or Ruby), but without extending any of the built-in JavaScript objects. It's the tie to go along with jQuery's tux, and Backbone.js's suspenders.

Underscore provides 60-odd functions that support both the usual functional suspects: map, select, invoke ? as well as more specialized helpers: function binding, javascript templating, deep equality testing, and so on. It delegates to built-in functions, if present, so modern browsers will use the native implementations of forEach, map, reduce, filter, every, some and indexOf.

Collection Functions

each

The cornerstone, an 'each' implementation, aka 'forEach'.
Handles objects with the built-in 'forEach', arrays, and raw objects.

Collection Functions

map

Return the results of applying the iterator to each element.

Collection Functions

reduce

builds up a single result from a list of value, aka 'inject', or 'foldl'.

Collection Functions

reduceRight

The right-associative version of reduce, also known as 'foldr'.

Collection Functions

find

Return the first value which passes a truth test. Aliased as 'detect'.

Collection Functions

filter

Return all the elements that pass a truth test.

Collection Functions

reject

Return all the elements for which a truth test fails.

Collection Functions

all

Determine whether all of the elements match a truth test.

Collection Functions

any

Determine if at least one element in the object matches a truth test.

Array Functions

first

Get the first element of an array.
Passing **n** will return the first N values in the array.

signature : function(array, n, guard)

Array Functions

initial

Returns everything but the last entry of the array.
Especcialy useful on the arguments object.
Passing **n** will return all the values in the array, excluding the last N.

signature : function(array, n, guard)

Array Functions

last

Get the last element of an array.
Passing **n** will return the last N values in the array.

signature : function(array, n, guard)

Array Functions

rest

Returns everything but the first entry of the array.
Especially useful on the arguments object.
Passing an **index** will return the rest of the values in the array from that index onward.

signature : function(array, index, guard)

Array Functions

compact

Trim out all falsy values from an array.
Returns a copy of the array with all falsy values removed.
In JavaScript, false, null, 0, "", undefined and NaN are all falsy.

signature : function(array)

Array Functions

flatten

Return a completely flattened version of an array.
Flattens a nested array (the nesting can be to any depth).
If you pass shallow, the array will only be flattened a single level.

signature : function(array, shallow)

Array Functions

without

Return a version of the array that does not contain the specified value(s).
Returns a copy of the array with all instances of the values removed. === is used for the equality test.

signature : function(array, [*values])

Array Functions

union

Produce an array that contains the union: each distinct element from all of the passed-in arrays.
Computes the union of the passed-in arrays: the list of unique items, in order, that are present in one or more of the arrays.

signature : function()

Array Functions

intersection

Produce an array that contains every item shared between all the passed-in arrays.
Computes the list of values that are the intersection of all the arrays.
Each value in the result is present in each of the arrays.

signature : function(array)

Array Functions

difference

Take the difference between one array and a number of other arrays.
Only the elements present in just the first array will remain.
Similar to without, but returns the values from array that are not present in the other arrays.

signature : function(array)

Array Functions

uniq

Produce a duplicate-free version of the array.
If the array has already been sorted, you have the option of using a faster algorithm.

signature : function(array, isSorted, iterator)

Array Functions

zip

Zip together multiple lists into a single array -- elements that share an index go together.

signature : function()

Array Functions

indexOf

Returns the index at which value can be found in the array, or -1 if value is not present in the array.
Uses the native indexOf function unless it's missing.
If you're working with a large array, and you know that the array is already sorted, pass true for isSorted to use a faster binary search.

signature : function(array, item, isSorted)

Array Functions

lastIndexOf

Returns the index of the last occurrence of value in the array, or -1 if value is not present.
Uses the native lastIndexOf function if possible.

signature : function(array, item)

Array Functions

range

A function to create flexibly-numbered lists of integers, handy for 'each' and 'map' loops.
'start', if omitted, defaults to 0;
'step' defaults to 1.
Returns a list of integers from 'start' to 'stop', incremented (or decremented) by 'step', exclusive.

signature : function(start, stop, step)
